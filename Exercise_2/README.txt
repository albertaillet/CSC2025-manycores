If you are not already in a SWAN session, do the following:
- Create a cernbox if you don't have one: https://cernbox.cern.ch
- Start a SWAN session at https://swan.cern.ch The default configuration is fine.
- Click on the little cloud icon at the top (old interface) or the git symbol in the left sidebar (new interface)
  to clone the following repository:
  https://github.com/hageboeck/CSC_Spark_Exercise.git
